Simple Linear Regression

To develop a better understanding the role of the slope and intercept in defining a regression line and to observe how coefficient estimates influence performance measures, explore the below interactive activity. This is an interactive scatterplot that illustrates the relationship between screen time duration and sleep duration. The line in red is the line of best fit, accompanied by the RSE and r-squared values. The line in blue is customisable. You can adjust the slope and intercept and derive different line equations. You can compare the different lines you generate with the red line of best fit. Have a look at how the r-squared and RSE values change for different line equations.

# Sample data
set.seed(123) # For reproducibility
data <- data.frame(
  years_of_education = runif(100, 0, 20), # Years of education from 0 to 20
  annual_income = 30 + runif(100, 0, 20) * 2.5 + rnorm(100, 0, 20) # Annual income
)

# Create a Plotly plot
p <- plot_ly(data, x = ~years_of_education, y = ~annual_income, type = 'scatter', mode = 'markers',
             marker = list(size = 10, color = 'rgba(255, 182, 193, .9)', line = list(color = 'rgba(152, 0, 0, .8)', width = 2)),
             text = ~paste('Years of Education: ', years_of_education, '<br>Annual Income: ', annual_income)) %>%
  layout(title = 'Years of Education vs. Annual Income',
         xaxis = list(title = 'Years of Education'),
         yaxis = list(title = 'Annual Income (in thousands)'),
         hovermode = 'closest')

# Display the plot
p
